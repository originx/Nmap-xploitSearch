xploitSearch v 1.0

Nmap script for detecting exploits on remote hosts

Get data from scanned ports and services with OS fingerprint (if exists) and query exploitsearch.net to get possible number of exploits on specific service.

Usage: nmap -sC -O ipLocation -script xploitSearch.nse --script exploit-search.nse [--script-args detailed= forcedOS= exploitsOnly= ]

For detailed usage check script source (LuaDoc)

DISCLAIMER:

Number of shown exploits doesn't reflect possible number of security holes that exists on your server. exploitsearch.net crawls possible security databases and depending on the quality of the query returns different results. Contributors:
